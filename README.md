# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/BeyondTeam/BDReborn.git
cd BDReborn
chmod +x beyond.sh
./beyond.sh install
./beyond.sh 
# Enter a phone number & confirmation code.

# For Auto Launch:
cd BDReborn
chmod 777 autobd.sh
screen ./autobd.sh
```
